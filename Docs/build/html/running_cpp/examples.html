Example input files¶

This section allows you to download input files that correspond to different physical situations. For a definition of the different parameters that are set in these files, see the section Input parameters. For a complete list of all example input files, have a look at our Examples/ directory. It contains folders and subfolders with self-describing names that you can try. All these input files are automatically tested, so they should always be up-to-date.
